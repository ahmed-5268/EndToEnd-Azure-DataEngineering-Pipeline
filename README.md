# EndToEnd-Azure-DataEngineering-Pipeline

## Overview
Built a scalable data pipeline using Azure services to ingest, process, and analyze data.

## Tools & Technologies
- Azure Data Factory
- ADLS Gen2
- Databricks (PySpark)
- Azure Synapse
- SQL

## Architecture
ADF → ADLS → Databricks → Synapse → Power BI

## Features
- Incremental data loading
- Medallion Architecture (Bronze, Silver, Gold)
- Partitioned data storage
- Distributed data processing using PySpark

## Output
- Clean and structured data for analytics
- Dashboard-ready datasets
